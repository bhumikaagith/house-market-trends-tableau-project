How this was built

01 — Data

Clean, transformed dataset

21,609 residential sales with price, size, age, renovation, and location fields — already cleaned with categorical fields one-hot encoded.

02 — Tableau

Calculated fields + 8 visuals

Eight calculated fields collapse the one-hot columns into readable dimensions, feeding a responsive dashboard and a five-point story.

03 — Web

Embedded with Flask

The published Tableau Public dashboard and story are embedded directly into this small Flask app, with live KPIs computed from the source CSV.
